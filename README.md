# HR Employee Attrition Analysis

## Overview
End-to-end HR analytics project identifying key 
factors driving employee attrition using Python 
statistical analysis and Tableau dashboard.

## Dataset
- IBM HR Analytics Employee Attrition dataset
- 1,470 employees | 35 variables
- Source: Kaggle (real IBM dataset)

## Tools Used
- Python: Pandas, NumPy, Matplotlib, Seaborn, SciPy
- Tableau Public (interactive dashboard)
- Jupyter Notebook (Anaconda)

## Analysis Performed
- Data cleaning and quality check
- 6 Python visualisations
- Statistical t-tests (SciPy)
- Correlation heatmap (10 variables)

## Dashboard
- 3 KPI cards (Total Employees, Left, Attrition Rate)
- 4 bar charts (Department, Overtime, Salary, Job Role)
- 2 interactive filters (Department, Attrition)
- Tableau Public:[[(https://public.tableau.com/app/profile/asmita.katake)]]


## Key Findings
1. Attrition Rate: 16.12%
2. Employees Left: 237 out of 1,470
3. Sales dept highest attrition: 133 employees left
4. Overtime employees show higher attrition (127 vs 110)
5. Low salary slab has highest attrition (163 employees)
6. Income difference statistically significant (p < 0.05)
